    • AlgoLearn

    As my final year project, I developed "AlgoLearn," an Android application designed to support new computer science students in mastering algorithms through an interactive learning experience. This project represented my first foray into mobile application development, providing me with the opportunity to learn and apply Android development techniques from the ground up.

    The application features a variety of interactive tools, including algorithm visualizations, step-by-step guides, and quizzes that reinforce the learning material. The goal was to make complex concepts more accessible to beginners by allowing them to see algorithms in action and practice their knowledge in a hands-on manner.

    Throughout the development process, I familiarized myself with key technologies such as Android Studio, Java, and XML for designing the user interface. I also implemented various Android components, such as activities, fragments, and view models, to create a seamless and intuitive user experience. Additionally, I integrated educational content and assessments to provide a comprehensive learning environment within the app.

    This project was both challenging and incredibly rewarding. As someone new to Android development, I faced a steep learning curve but was able to overcome it through persistence and a passion for learning. The experience not only enhanced my technical skills in mobile development but also deepened my understanding of how to design educational tools that effectively engage users. "AlgoLearn" is a testament to my ability to take on new challenges, learn rapidly, and deliver a functional and impactful software solution.
