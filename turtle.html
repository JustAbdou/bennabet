    • Building an Intelligent Robot System Using ROS and Machine Learning

    As part of a module in Robotics, I had the opportunity to work on an exciting project involving the TurtleBot. Throughout this project, I learned to work with the Robot Operating System (ROS), which was integral to the robot's functionality. The objective was to create an intelligent robot system using machine learning techniques that could navigate into a room, identify whether the room contained an image of Earth or the Moon, and distinguish between other planetary objects.

    Motion Planning and Efficient Room Exploration

    A critical aspect of the project was programming an efficient motion planning algorithm for the robot. The task required the robot to visit every part of the room within a strict 3-minute time limit, so it was essential to optimize the robot's movements. Through careful planning and programming, we ensured that the robot could explore the room efficiently and perform its identification tasks within the time constraints.

    This project gave me hands-on experience with key technologies, including ROS, Python, and machine learning techniques. I was able to deepen my understanding of robotic systems, from programming low-level robot control to implementing high-level decision-making algorithms. Through trial and error, I honed my problem-solving skills, learning how to troubleshoot complex issues and optimize performance under pressure.

    Working on the TurtleBot project was a challenging yet rewarding experience. It not only enhanced my technical abilities but also strengthened my teamwork and project management skills. The experience taught me the value of integrating different systems—ranging from hardware control to machine learning algorithms—to create a fully functional and intelligent robot. It also inspired me to continue exploring the fascinating field of robotics.
